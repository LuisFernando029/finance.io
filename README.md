Projeto de finalização de semestre da matéria de Técnicas de Programação

## Tecnologias utilizadas

Hibernate <br>
Postgres <br>

# Endpoints

## Transações 
Para criar uma transação - [POST] "/transacao" <br>
Para selecionar todas as transações - [GET] "/transacao" <br>
Para selecionar uma transação - [GET] "/transacao/{id}" <br>
